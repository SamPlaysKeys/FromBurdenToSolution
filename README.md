# From Burden to Solution: Reframing Multifactor Authentication in Contemporary Cybersecurity

## Overview
This repository contains public resources for a presentation focused on helping technical professionals communicate Multifactor Authentication (MFA) effectively to non-technical audiences. The materials aim to reframe MFA from a perceived burden into a practical, value-driven solution that enhances security posture while improving user experience.

## Presentation Goals
- **Redefine multifactor authentication** for both professionals and the public by clarifying benefits, risks, and modern best practices
- **Present dynamic approaches** for pitching and delivering MFA, including messaging frameworks tailored to different audiences
- **Empower cybersecurity professionals** to positively shift public perception of MFA as individuals, community members, and experts

## Repository Contents
- **Presentation Overview**: `Full Copy - From Burden to Solution - Reframing Multifactor Authentication in Contemporary Cybersecurity.md` - Complete session outline and narrative flow
- **Slide Deck**: `Reframing Multifactor Authentication - Simply Cyber Con.pptx` - Visual presentation materials
- **Planning Notes**: `General MFA Planning Notes.md` - Facilitator prompts, drafts, and supporting notes
- **Docs/**: Research papers and supporting materials referenced in the presentation

## Target Audience
Technical practitioners who must explain MFA to stakeholders and end users, including:
- IT and cybersecurity professionals
- System administrators and identity architects
- Technical leaders preparing enablement content for non-technical audiences
- Anyone interested in improving MFA adoption and communication strategies

## Key Themes
- **Sales-style communication strategies** for technical topics
- **Adult learning concepts** applied to security training
- **Empathetic messaging** that acknowledges user frustrations
- **Real-world case studies** demonstrating successful MFA adoption
- **Community-driven approaches** to changing security culture
